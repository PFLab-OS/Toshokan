# Toshokan

## Requirements

* docker (tested on 18.06.0-ce)
* scons (> 3.0)
* python (> 2.5)

## Install & Test
```
$ git clone https://github.com/PFLab-OS/Toshokan.git
$ cd Toshokan/hakase
$ scons prepare
$ scons
```

The message (`All tests have successfully finished!`) will be shown if self-check tests are passed.
